# 认知疫苗 · 原型文档

> 这是一次认知疫苗接种实验。
> 它不是一个产品。它是一个"事先警告"——让受众更机敏。

## 核心哲学

- **不是妖魔化任何App** — 抖音、B站、小红书不是敌人。它们只是做了它们该做的事
- **不是贩卖焦虑** — "这不是你的问题，这说明你的大脑是正常的"
- **不是高高在上的教育** — 设计者本人就是被设计过的其中之一
- **不是一次性能解决所有问题** — 这是第一针，后续加强在用户自己手里

### 第一性原理

我们不是在制造另一个"让人上瘾"的东西。
我们是在给人们一个选择——知道自己被什么带走，然后决定要不要继续被带走。

## 交互流程

```
[首屏：复述体验 → 正常化]
    ↓
[第一个选择：直面 vs 回避]
    ├── 选"随便点进来" → "下一站"分支 → 汇入主流程
    └── 选"我想看看" → 直接进入主流程
    ↓
[揭示：我们都是被设计过的]
    ↓
[多巴胺机制解释 → "你的大脑在被训练"]
    ↓
[30秒静坐倒计时 → 体验"安静"]
    ↓
[元认知 → 两个入口（书 / 大模型）]
    ↓
[感谢 → "你已经不一样了"]
    ↓
[邀请转发：为你在乎的人]
    ↓
[背后的故事 → GitHub参与入口]
```

## 交互点明细

| 环节 | 类型 | 说明 |
|------|------|------|
| 首屏 | 被动阅读 | 文字渐出，复述用户可能有的体验 |
| 第一个选择 | 主动选择 | 两个按钮，大小一致，无主次 |
| "下一站"分支 | 主动选择 | 6个App，用户选任一后揭示"同样被设计" |
| 多巴胺揭示 | 点击揭示 | 点按钮显示关键句 |
| 30秒倒计时 | 被动体验 | 全屏倒计时，无干扰 |
| 入口推荐 | 被动阅读 | 两个入口卡片 |
| 感谢 | 被动阅读 | 有人直接对用户说的话 |
| 转发邀请 | 主动行动 | 复制链接 / 查看背后故事 |
| 参与入口 | 被动阅读 | GitHub仓库链接 |

## 状态管理

- 无后端 — 纯前端H5页面
- 无用户登录
- 无数据采集
- URL可包含 `?from=xxx` 追踪来源（手动）

## 技术方案

### 推荐
- 单HTML文件 + 纯CSS + 原生JS
- 部署到 GitHub Pages / Vercel / Netlify / 任何静态托管

### 最小可行
- 一个能双击打开的HTML文件
- 不需要服务器
- 不需要API

### 后续可扩展方向
- 接入文案JSON（外置配置文件实现多语言/多受众版本）
- 主题切换（浅色/深色/大字版）
- 转发链追踪（URL参数）
- A/B测试不同文案版本

## 核心原则

1. **语言降维** — 买菜阿姨能听懂
2. **不妖魔化** — 算法不是敌人，无意识才是
3. **不制造新束缚** — 用户看完后应该感到被交付了一个方向，不是被教育了一顿
4. **真诚** — 结尾是真实的人写的，不是品牌文案
5. **低门槛** — 一个链接就能传，不用下载、不用注册

## 种子版本定位

这个原型的第一目标是：**被第一个觉醒者看到 → 被转发 → 被有能力迭代的人拿到 → 被改进**。

- 它不需要覆盖所有人
- 它不需要完美
- 它只需要让对的人看到，然后说"这个我可以帮忙"
